# Basalt Issue Tracker

Public issue tracker for [Basalt](https://basalt.host).

This repository is used to report bugs, request features, and track known issues for Basalt. The source code itself lives in a private repository; this repo exists so anyone can open and follow issues.

## Reporting an issue

1. [Search existing issues](https://github.com/basalt-host/issues/issues?q=is%3Aissue) first, since your problem may already be reported.
2. Open a [new issue](https://github.com/basalt-host/issues/issues/new/choose) using the appropriate template:
   - **Bug report**: something isn't working as expected
   - **Feature request**: suggest an idea or improvement
   - **Template request**: request a new game template

When reporting a bug, please include your Basalt version, your platform/OS, and clear steps to reproduce. The more detail you provide, the faster we can help.

## What doesn't belong here

- **Security vulnerabilities**: please do not open a public issue. Contact us privately instead.
- **General questions / support**: check the [documentation](https://github.com/basalt-host/docs) first.
